Contexte

Projet personnel développé en Python pour la salle de sport Hall b (Saint-Dionisy), où j'ai effectué mon stage. L'objectif : identifier l'emplacement géographique optimal pour leur implantation, en croisant deux sources de données publiques.

Le score bi-critères combine la densité de population INSEE (pondérée par noyau gaussien) et la pression concurrentielle issue des salles de sport OpenStreetMap (pénalité exponentielle).

Démo — Tester le logiciel

Sélectionnez une ville, ajustez les paramètres et lancez l'analyse pour voir les zones optimales s'afficher sur la carte en temps réel.

Pipeline
1
Collecte des données
Données démographiques par commune (INSEE) et extraction des salles de sport existantes via l'API Overpass d'OpenStreetMap.
pandas · overpy · insee api
2
Cartographie de la concurrence
Calcul de la densité de concurrence autour de chaque point candidat via la formule de distance haversine.
overpy · geopy · haversine
3
Calcul du score bi-critères
score = pop_pondérée × pénalité_concurrence
numpy · gaussian kernel
4
Visualisation
Génération d'une carte de chaleur des scores et identification de l'emplacement optimal.
matplotlib · heatmap
Extrait de code
optimisation.py
import numpy as np from geopy.distance import geodesic SIGMA_POP = 1.5 # rayon gaussien population (km) SIGMA_CONCUR = 0.8 # rayon pénalité concurrence (km) def population_ponderee(point, communes): total = 0.0 for c in communes: dist = geodesic(point, c.coords).km poids = np.exp(-dist**2 / (2 * SIGMA_POP**2)) total += c.population * poids return total def penalite_concurrence(point, salles_osm): pression = 0.0 for s in salles_osm: dist = geodesic(point, s.coords).km pression += np.exp(-dist**2 / (2 * SIGMA_CONCUR**2)) return np.exp(-pression) def score(point, communes, salles_osm): return population_ponderee(point, communes) * penalite_concurrence(point, salles_osm)

Apports

Ce projet m'a permis de structurer un programme Python complet de bout en bout, de l'import des données publiques à la génération d'une carte de score exploitable.

J'ai développé des compétences en modélisation mathématique (noyaux gaussiens, pénalité exponentielle), en manipulation de données géospatiales et en visualisation. Ce type d'approche est directement transposable à des problématiques d'ingénierie comme l'implantation d'unités de production.
