⚠️ Serveur local requis

Ce fichier doit être servi via un serveur HTTP local pour que les salles de sport puissent être chargées. Ouvrez un terminal dans ce dossier et lancez :

python -m http.server 8765

Puis ouvrez votre navigateur sur :

http://localhost:8765/gymlocator.html

Python est préinstallé sur Mac et Linux. Sur Windows, téléchargez-le sur python.org.
Ou double-cliquez sur lancer_windows.bat si vous êtes sur Windows.
